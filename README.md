# Проект: Место

## Описание
Сервис Место: одностраничный, адаптивный сайт, представляющий собой интерактивную страницу, куда можно добавлять фотографии, удалять их и ставить лайки. 

### Стек
* HTML 5;
* CSS 3;
* JavaScript;
### Технологии
* Flexbox;
* Grid Layout;
* Media queries;
* Nested БЭМ;
* Pop-up;
